3.1 How We Build Aumazing

Our 5 Steps (Kanban Board):

➔

➔
🔨 Doing

➔

Figure 3.1: Kanban Board

Why this works:

  • ✓ Can change things while building
  • ✓ Test each part as we go
  • ✓ See progress clearly
  • ✓ Focus on one thing at a time
📱

3.2 App Technology

🎮 Flutter + Flame

For making games and screens that work on phones and tablets!

🗄️ Supabase

For storing all the important data safely in the cloud!

💾 SQLite

For playing games even when there's no internet!

🤖 Python + FastAPI + XGBoost

For the smart AI that checks how kids are doing!

📹 Jitsi Meet

For video calls with therapists (Premium feature)!

👨‍👩‍👧

3.3 User Requirements

What parents need:

  • 🔐 Sign up and log in safely
  • 👶 Create a profile for their child
  • 📊 See their child's progress
  • ⏰ Set screen time limits
  • 📹 Connect with therapists (Premium)

What children need:

  • 🎮 Fun, colorful games
  • 📷 Clear visual guides
  • 🏆 Rewards for completing tasks
🎯

3.4 Features List

Core Features:
  • User registration/login
  • Child profile management
  • Pre-assessment quiz
  • 2D learning games
  • Level progression
  • Post-assessment
  • Parent dashboard
Special Features:
  • AI assessment (XGBoost)
  • Smart recommendations
  • Screen time control
  • Progress tracking
  • Offline play (SQLite)
  • Therapy directory
  • Video conferencing
💻

3.5 Hardware Needed

For Development:

  • 💻 Laptop/computer
  • 📱 Android phone/tablet for testing
  • 🌐 Internet connection

For Users:

  • 📱 Android or iOS phone/tablet
  • 🌐 Internet (for most features)
  • 💾 Storage space for the app
✅

3.6 Is This Project Possible?

💰 Economically: Yes! Using free tools and student resources.

🔧 Technically: Yes! All needed technologies exist and work well together.

👨‍👩‍👧 For Families: Yes! It directly helps children with autism learn at home.

📅 On Time: Yes! The 6-month plan (Feb-Sep 2026) is doable.
